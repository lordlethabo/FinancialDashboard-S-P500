# Financial Dashboard & Insights on S&P500 Companies

**AI-powered financial analysis project** that provides actionable insights on S&P500 companies.  
Built with **Python, Pandas, Matplotlib/Seaborn/Plotly**, and **LLaMA 2** for textual summaries, this project visualizes key financial metrics, assesses company risk levels, and demonstrates applied AI for finance.  
**Oracle AI Foundation Associate certified.**

---

## Problem
Investors and startups need **quick insights into company performance and risk levels**. This project analyzes historical S&P500 financial data to provide actionable insights for decision-making.

## Tools & Skills
- Python: Pandas, NumPy
- Visualization: Matplotlib, Seaborn, Plotly
- AI/LLM: LLaMA 2 (optional for summaries)
- Financial Analysis: Revenue, Net Income, EBIT, Cost of Revenue
- Oracle AI Foundation Associate certification

## Workflow
1. Load and clean data (`data/sp500_financial_data.csv`)
2. Analyze key financial metrics
3. Assess company risk (Safe / High Risk)
4. Create interactive dashboard visualizations
5. Optional: Generate company summaries using LLaMA 2 prompts

## How to Run
1. Clone repo: `git clone https://github.com/lordlethabo/FinancialDashboard-S&P500.git`
2. Open `notebooks/financial_dashboard.ipynb` in Jupyter Notebook or Google Colab
3. Ensure `data/sp500_financial_data.csv` is in the `data/` folder
4. Run all cells to generate dashboards and insights
